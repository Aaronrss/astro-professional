## ¿Por qué elegir Astro?

Astro ofrece varias ventajas sobre otros frameworks:

1. **Rendimiento excepcional** - Envía cero JavaScript por defecto
2. **Flexibilidad** - Usa cualquier framework UI que prefieras
3. **Experiencia de desarrollo** - Hot module replacement y TypeScript integrado
