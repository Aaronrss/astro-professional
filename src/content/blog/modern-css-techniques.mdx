## Container Queries

Las container queries permiten estilos basados en el contenedor padre:

```css
.card-container {
  container-type: inline-size;
}

@container (min-width: 400px) {
  .card {
    display: flex;
  }
}
```

## CSS Nesting

El anidamiento nativo de CSS finalmente está aquí:

```css
.button {
  background: blue;
  
  &:hover {
    background: darkblue;
  }
  
  & .icon {
    margin-right: 0.5rem;
  }
}
```
